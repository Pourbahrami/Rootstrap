<h3 align="center">Rootstrap</h3>

<p align="center">
  Rootstrap is a Sass port for RTL version of <a href="https://github.com/twbs/bootstrap">Bootstrap 3</a>
</p>


<br>

## Installation

```console
$ npm install rootstrap
```

## Use in Laravel

```scss
// Bootstrap
@import "~rootstrap/assets/scss/rootstrap";
```
